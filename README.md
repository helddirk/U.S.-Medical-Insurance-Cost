# U.S.-Medical-Insurance-Cost
This is a project about the U.S Medical Insurance Data. It is part of the “Data Scientist Career path” from Codecademy.com. It is part of the first part of the foundations. The requirements were to use Python. At this stage there was only the “csv” library introduced. Which means that I did not use other libraries.
